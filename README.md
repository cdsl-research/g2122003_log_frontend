# 書籍レビューサイト　フロントエンドのリポジトリ

Reactにより作成されており、Dockerfileでコンテナ化されます。

# ローカルでの実行方法

## 実行

0. dockerをインストールしておく
1.githubからクローンする。

2.以下のコマンドを実行
```
cd g2122003_log_frontend 
docker-compose up
```

3.ブラウザからlocalhostにアクセス

## 終了
```
docker-compose down
```
